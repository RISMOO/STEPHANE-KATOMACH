PORTROFF - Documentation


Portroff is a bootstrap landing page theme for you to promote your work, the theme has a fully responsive technology, that makes your website fit any screen size, positively impacting the accessibility of the content, the theme also has an elegant and minimalist design making it easy to present your portfolio to the visitor.

Created: 03/11/2021

By: Dimitroff

What we need to start


  1. Unzip the template file;
  2. Install code editor like vs code;
  3. Open the project in the code editor;
  4. Open index.html on your web browser to see your website.

Features


  • Font Awesome;
  • Google Fonts;
  • Bootstrap 5;
  • Jquery;
  • Owl-Carousel;
  • AOS - Animate on scroll library.

Background Image and Color


/*Theme Setup*/
:root {
	/*COLOR VAR*/
  --main-color: #d94f14;
  /*BACKGROUND IMAGE*/
  --section_1-background:  url(URL_IMAGE_HERE);
  --section_5-background: url(URL_IMAGE_HERE);
}

Change the default theme color by hovering over items:

  1. open the style.css located inside the css folder at the root of the file;
  2. Find the variable (--main-color) at the beginning of the file;
  3. change the hexadecimal value (the default value is #d94f14).
    This way the colors of the links and some of the items will be changed when hovering over them.

Change the background images from section 1 and section 5:

  1. open the style.css located inside the css folder at the root of the file;
  2. Find the variable (--section_1-background) at the beginning of the file;
  3. Add the path of the image inside the parentheses;
  4. Repeat the previous step for the image in section 5. (--section_5-background).

  1. Access the video on youtube;
  2. Click share and then embed;

  4. Paste the link inside the data-video tag;
